Memoization

memoize([delay], fn)
Wrap a function to store its results for fast access via identical arguments.

Parameters:

  • delay int number of updates that should pass before data is cleared; use 0/false/nil to never clear (optional)
  • fn func function to be memoized; should only take stringable arguments and return a non-nil value
reset_data(update_count)
Clear outdated data gathered via util.memoize. Call this once per update cycle.

Parameters:

  • update_count int conky's $update_count
